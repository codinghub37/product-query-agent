# 🛍️ Product Query Agent

An AI-powered Product Query Agent that enables users to search and retrieve product information using natural language. The project leverages LLM-based techniques to understand user queries and provide relevant product recommendations.

---

## 🚀 Features

* Natural language product search
* AI-powered query understanding
* Retrieve relevant product information
* Easy-to-use Python implementation
* Beginner-friendly project

---

## 🛠️ Tech Stack

* Python
* LangChain
* Groq LLM
* Pandas
* Jupyter Notebook

---

## 📁 Project Structure

```text
product-query-agent/

└── README.md
```

---

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/codinghub37/product-query-agent.git
```

### Navigate to the Project

```bash
cd product-query-agent
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file and add your API key:

```env
GROQ_API_KEY=your_api_key_here
```

---

## ▶️ Run the Project

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open `product_query_agent.ipynb` and run all cells.

---

## 📚 Workflow

1. Load the product dataset.
2. Process user queries written in natural language.
3. Analyze the query using an LLM.
4. Retrieve the most relevant product information.
5. Return an intelligent response to the user.
